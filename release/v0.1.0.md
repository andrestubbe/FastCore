## ✨ Features

### Unified Native Library Loader
- **Automatic DLL/SO Extraction**: Bundled native libraries are extracted from JAR resources to a temporary directory at runtime — zero manual setup required.
- **Class-Context Aware**: Resolves resources relative to the calling class for correct module isolation across complex classpaths.
- **Deduplication Cache**: Tracks already-loaded libraries to avoid redundant disk I/O on repeated calls.
- **Platform Detection**: Detects OS (Windows / Linux / macOS) and architecture (x86_64 / ARM64) at runtime.
- **Ecosystem Base**: The mandatory foundation for all FastJava native modules (FastAudioPlayer, FastTTS, FastSTT, etc.).

## 📦 Installation

### Maven (JitPack)
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>com.github.andrestubbe</groupId>
        <artifactId>fastcore</artifactId>
        <version>v0.1.0</version>
    </dependency>
</dependencies>
```

### Gradle (JitPack)
```groovy
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.andrestubbe:fastcore:v0.1.0'
}
```
