## 📌 Title
FastCore for Java — The Zero‑Overhead Core Library

## 📝 Description
FastCore is the ultra-minimal, zero-overhead core library powering the entire FastJava ecosystem.
It provides the fundamental building blocks used by all Fast* modules — designed for raw speed and predictable native behavior.

### 🚀 Key Features:
- **Zero-Allocation**: Minimal GC pressure via optimized utilities.
- **Foundation Layer**: Powers FastCamera, FastDisplay, FastTheme, and more.
- **Native Integration**: Specialized for JNI library loading and extraction.
- **Zero Dependencies**: Keeps your project lean and clean.

### 🔥 Use Cases:
- The base layer for all high-performance FastJava modules.
- Standalone utilities for performance-critical Java code.
